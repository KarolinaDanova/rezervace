Rezervace termínu

Vyberte volný termín a vyplňte kontaktní údaje

⚙️ Před spuštěním: připoj Supabase databázi
Najdi v kódu sekci KONFIGURACE a doplň:
  • SUPABASE_URL — z Supabase projektu → Settings → API
  • SUPABASE_ANON_KEY — tamtéž
→ Zaregistruj se zdarma na supabase.com

SQL pro vytvoření tabulky (Supabase → SQL Editor):
CREATE TABLE rezervace (
  id SERIAL PRIMARY KEY,
  termin TEXT NOT NULL,
  jmeno TEXT NOT NULL,
  email TEXT NOT NULL,
  telefon TEXT,
  poznamka TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);
